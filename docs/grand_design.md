# 馬券会議 グランドデザイン

## プロダクト思想

> **「依存症傾向のあるユーザーが、AIとの対話を通じて立ち止まって考え、自己設定した負け額限度額内で責任ある賭けを行える」**

このプロダクトは**競馬予想アプリではない**。

- 勝ち方を教えるアプリ → **NO**
- 衝動的な購入を促すアプリ → **NO**
- 「当たる」を煽るアプリ → **NO**

「立ち止まって、考えましょう」を具現化するアプリである。

## デザイン4原則

### 原則1: 煽らない

| 一般的な競馬アプリ | 馬券会議 |
|-------------------|---------|
| 「今すぐ買え！」 | 「立ち止まって、考えましょう」 |
| 派手な色・アニメーション | 落ち着いた色・静かな動き |
| 大きなCTAボタン | 控えめなアクション |
| 緊急感を煽る | 余裕を与える |
| オッズを強調 | データを淡々と提示 |

### 原則2: 立ち止まらせる

- 購入ボタンを押す前に**必ずAI相談を経由**
- 衝動的なタップを防ぐ**適度な摩擦**
- 「やめておく」という選択肢を**常に用意**

### 原則3: 味方であることを示す

- AIは「煽る存在」ではなく「寄り添う存在」
- 「あなたの代わりに考える」ではなく「あなたと一緒に考える」
- 責める口調ではなく、共感する口調

### 原則4: 現実を直視させる

- 損益を**隠さず、ごまかさず**表示
- 負け額限度額を**常に意識させる**
- 「いくら使ったか」を**客観的に提示**

## ボタンの優先順位

### 「やめておく」が主アクション

```
┌────────────────────────────────────┐
│                                    │
│    ┌────────────────────────┐      │
│    │                        │      │
│    │    やめておく          │      │  ← 大きく、安心感のある色（緑）
│    │                        │      │
│    └────────────────────────┘      │
│                                    │
│         購入する                   │  ← 控えめ、テキストリンク風
│                                    │
└────────────────────────────────────┘
```

「やめておく」を選ぶことが**ポジティブな行動**だと感じさせる。
「購入する」は**慎重に選ぶべき選択肢**だと伝える。

## カラーパレット

```css
:root {
  /* 基調色: 落ち着き・安心 */
  --color-calm-green: #1a5f2a;      /* 信頼・安定・自然 */
  --color-calm-green-light: #e8f5ea; /* 背景・選択状態 */

  /* 警告色: 注意喚起（煽りではなく気づき） */
  --color-alert-amber: #f59e0b;     /* 限度額接近時 */
  --color-alert-red: #dc2626;       /* 限度額到達時 */

  /* 中立色: データの客観的表示 */
  --color-neutral-50: #f9fafb;
  --color-neutral-200: #e5e7eb;
  --color-neutral-500: #6b7280;
  --color-neutral-800: #1f2937;
}
```

**注意**: 赤・オレンジは「購入を促す」ためではなく、「危険を知らせる」ためにのみ使用。

## 常時表示すべき要素

### 1. 負け額限度額バー（将来実装予定）

すべての画面で常に表示される「限度額バー」

- 通常時: 緑
- 80%到達: 黄色（警告）
- 100%到達: 赤（購入停止）

> **Note**: ログイン機能実装後に対応予定。現在はカート画面にモックを表示。

### 2. 依存症相談窓口リンク

すべてのページで**ボトムナビの上**に常時表示

```
困ったときは｜ギャンブル依存症相談窓口 →
```

- 目立ちすぎない（アプリの邪魔にならない）
- しかし、必要なときに**すぐ見つけられる**

## やってはいけないこと

| NG | 理由 |
|---|------|
| CTAボタンを目立たせる | 購入を煽ることになる |
| カードに立体感・影を強調 | 視覚的刺激を増やす必要はない |
| ゴールドアクセントで華やかに | 競馬アプリ的な煽りになる |
| ホバー効果を派手に | 衝動的クリックを誘発する |

**このアプリは「かっこいいUI」を目指すものではない。**

## 最優先ペルソナ

**田中 健一（42歳）**

- 競馬歴15年、月間10〜30万円（変動大）
- 負けを取り戻そうと賭け金が増加傾向
- 「やめたい」とは思っていないが、**「コントロールしたい」**と強く感じている
- 一人で賭けていると冷静な判断ができなくなる
- 家族に心配をかけたくない

このプロダクトのデザインは、**田中さんが冷静でいられるため**に存在する。

## まとめ

| 一般的なWebデザイン改善 | 馬券会議のデザイン改善 |
|------------------------|---------------------|
| 「より使いやすく」 | 「より立ち止まりやすく」 |
| 「より魅力的に」 | 「より冷静になれるように」 |
| 「コンバージョン向上」 | 「衝動的購入の抑制」 |
| 「ユーザーを惹きつける」 | 「ユーザーを守る」 |

**煽らない。急かさない。寄り添う。現実を見せる。**

それが馬券会議のデザイン哲学である。
