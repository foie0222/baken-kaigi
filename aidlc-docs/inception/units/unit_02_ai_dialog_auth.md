## US-01-007: 会話履歴の保存

**ストーリー**
> ログインユーザーとして、AIとの会話履歴を保存したい。なぜなら、過去の相談内容を振り返り、自分の賭けパターンを分析したいから。

| 項目 | 内容 |
|------|------|
| 優先度 | Could have |
| 認証 | **必要** |
| 対象ペルソナ | 全ペルソナ |
| 備考 | 保存期間: 1週間 |

**受け入れ条件**
- [ ] AI相談完了後、会話履歴が自動保存される
- [ ] 保存される内容は買い目、AIフィードバック、ユーザーの追加質問と回答
- [ ] 保存期間は1週間（7日経過後に自動削除）
- [ ] ログインユーザーのみ保存される
- [ ] 未ログインユーザーにはログイン促進メッセージが表示される

## US-01-008: 会話履歴の閲覧

**ストーリー**
> ログインユーザーとして、過去1週間のAIとの会話履歴を閲覧したい。なぜなら、過去の相談内容と結果を振り返りたいから。

| 項目 | 内容 |
|------|------|
| 優先度 | Could have |
| 認証 | **必要** |
| 対象ペルソナ | 全ペルソナ |

**受け入れ条件**
- [ ] 履歴一覧画面で過去1週間の会話が日付順に表示される
- [ ] 各履歴にレース名、買い目、相談日時が表示される
- [ ] 履歴をタップすると詳細（AIフィードバック、会話内容）が表示される
- [ ] 購入した場合は結果（当たり/外れ、払戻金）も表示される
- [ ] 履歴の削除機能がある
