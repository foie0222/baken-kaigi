# 馬券会議 - ユーザーペルソナ定義書

## 概要
本ドキュメントは、「馬券会議」アプリのターゲットユーザーを定義します。
ペルソナは優先順位順に記載されています。

---

## ペルソナ 1: 依存症傾向のあるユーザー（最優先）

### 基本プロフィール
| 項目 | 内容 |
|------|------|
| 名前 | 田中 健一（仮名） |
| 年齢 | 42歳 |
| 職業 | 会社員（営業職） |
| 家族構成 | 妻、子供2人 |
| 競馬歴 | 15年 |
| 月間賭け金額 | 10〜30万円（変動大） |

### 背景・状況
- 競馬が趣味で長年楽しんできたが、最近は負けを取り戻そうとして賭け金が増加傾向
- 給料日後に大きく賭けてしまい、月末に家計が苦しくなることがある
- 家族には賭け金額を正確に伝えていない
- 「やめたい」とは思っていないが、「コントロールしたい」と強く感じている

### 目標
- 競馬を長く楽しみ続けたい
- 毎月の負け額を自分で決めた範囲内に収めたい
- 衝動的な賭けを減らしたい
- 家族に心配をかけたくない

### 課題・ペインポイント
- 負けが続くと「次こそは」と思い、予定以上に賭けてしまう
- 一人で賭けていると冷静な判断ができなくなる
- 自分の賭けパターンや損益を正確に把握できていない
- 既存の予想アプリは「当たる」ことを煽り、かえって賭け金が増える

### ニーズ
- 自分で設定した負け額限度を絶対に超えられない仕組み
- 賭ける前に一度立ち止まって考える機会
- 客観的なデータに基づくフィードバック
- 損益の可視化による自己認識

### 利用シナリオ
1. レース前に買いたい馬券をアプリに入力
2. AIから「なぜその馬を選んだのか」などの質問を受ける
3. AIからデータに基づいたフィードバックを受け、冷静に判断
4. 今月の残り許容負け額を確認してから購入を決定
5. 月間の損益ダッシュボードで自分の賭けパターンを振り返る

## ペルソナ 2: 一般的な競馬ファン（娯楽目的のベッター）

### 基本プロフィール
| 項目 | 内容 |
|------|------|
| 名前 | 鈴木 大輔（仮名） |
| 年齢 | 35歳 |
| 職業 | ITエンジニア |
| 家族構成 | 独身 |
| 競馬歴 | 5年 |
| 月間賭け金額 | 2〜5万円（安定） |

### 背景・状況
- 週末の趣味として競馬を楽しんでいる
- 予算を決めて計画的に賭けている
- データ分析が好きで、自分なりの予想ロジックを持っている
- 依存症ではないが、より賢く賭けたいと思っている

### 目標
- 予想の精度を上げたい
- 自分の予想ロジックの妥当性を確認したい
- 長期的な収支をプラスに近づけたい
- 競馬をより深く楽しみたい

### 課題・ペインポイント
- 自分の予想が正しいのか客観的に判断できない
- 過去の賭け履歴を体系的に分析できていない
- 予想サイトは結果だけで、思考プロセスの改善に役立たない
- 購入時にIPATへの切り替えが面倒

### ニーズ
- 自分の予想に対する客観的なフィードバック
- 過去の賭けパターンと結果の分析
- シームレスな購入体験
- データに基づいた意思決定支援

### 利用シナリオ
1. 自分で予想した買い目をアプリに入力
2. AIから予想の根拠について質問を受け、自分の考えを整理
3. AIから関連データ（過去成績、馬場状態など）のフィードバックを受ける
4. 納得したらそのままIPAT連携で購入
5. 週末終了後、損益ダッシュボードで振り返り

## ペルソナ 3: 競馬初心者

### 基本プロフィール
| 項目 | 内容 |
|------|------|
| 名前 | 山田 美咲（仮名） |
| 年齢 | 28歳 |
| 職業 | 事務職 |
| 家族構成 | 独身 |
| 競馬歴 | 半年 |
| 月間賭け金額 | 5千円〜1万円 |

### 背景・状況
- 友人に誘われて競馬を始めた
- 馬券の買い方や予想の仕方がよくわからない
- 少額で楽しみたいが、どのくらいが適切かわからない
- ギャンブル依存症にならないか少し不安がある

### 目標
- 競馬の楽しみ方を学びたい
- 無理のない範囲で賭けを楽しみたい
- 予想の基本を理解したい
- 安全に競馬を始めたい

### 課題・ペインポイント
- 何を基準に馬を選べばいいかわからない
- 適切な賭け金額の目安がわからない
- IPATの使い方が複雑に感じる
- 依存症になるリスクが怖い

### ニーズ
- 予想の考え方を教えてくれるガイダンス
- 初心者向けの負け額限度額の提案
- シンプルで使いやすいインターフェース
- 安心して始められる環境

### 利用シナリオ
1. 気になった馬の名前をアプリに入力
2. AIから「どうしてその馬が気になったのか」を聞かれる
3. AIが初心者向けにデータの見方を説明
4. 少額から始められるよう負け額限度額を設定
5. 購入後、結果と振り返りを確認

## 設計への示唆

### 全ペルソナ共通
- AIとの対話による「立ち止まって考える」機会の提供
- データに基づく客観的なフィードバック
- IPAT連携によるシームレスな購入体験

### ペルソナ1向け（最優先で対応）
- 負け額限度額の厳格な強制（回避不可能な設計）
- 限度額接近時の目立つアラート表示
- 損益の明確な可視化

### ペルソナ2向け
- 詳細な分析機能
- 過去の賭けパターンとの比較
- 予想根拠の深掘り質問

### ペルソナ3向け
- 初心者向けのオンボーディング
- 用語説明・ガイダンス機能
- 推奨される負け額限度額の提案
