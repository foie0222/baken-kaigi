## US-01-001: レース一覧の表示

**ストーリー**
> 競馬ファンとして、当日開催されるレースの一覧を確認したい。なぜなら、どのレースに賭けるか検討したいから。

| 項目 | 内容 |
|------|------|
| 優先度 | Must have |
| 認証 | 不要 |
| 対象ペルソナ | 全ペルソナ |

**受け入れ条件**
- [ ] 当日開催の全レースが一覧表示される
- [ ] 各レースにレース番号、レース名、発走時刻が表示される
- [ ] 開催場（東京、中山、京都など）でフィルタできる
- [ ] 日付を選択して別日のレースも閲覧できる
- [ ] レースをタップすると詳細画面に遷移する
- [ ] 認証不要で閲覧できる

---

## US-01-002: レース詳細情報の表示

**ストーリー**
> 競馬ファンとして、特定のレースの出走馬、騎手、オッズなどの詳細情報を確認したい。なぜなら、賭ける馬を選ぶための情報が必要だから。

| 項目 | 内容 |
|------|------|
| 優先度 | Must have |
| 認証 | 不要 |
| 対象ペルソナ | 全ペルソナ |

**受け入れ条件**
- [ ] レース名、距離、馬場状態、発走時刻が表示される
- [ ] 出走馬一覧が馬番順に表示される
- [ ] 各馬の馬名、騎手名、オッズ、人気順が表示される
- [ ] 馬番は枠色で視覚的に識別できる
- [ ] 認証不要で閲覧できる

---

## US-01-003: 買い目の入力

**ストーリー**
> 競馬ファンとして、購入したい馬券の買い目（馬番、賭け式、金額）を入力したい。なぜなら、AIに相談する前に自分の賭け内容を明確にしたいから。

| 項目 | 内容 |
|------|------|
| 優先度 | Must have |
| 認証 | 不要 |
| 対象ペルソナ | 全ペルソナ |

**受け入れ条件**
- [ ] 券種（単勝、複勝、馬連、馬単、三連複、三連単）を選択できる
- [ ] 馬番をチェックボックスで選択できる
- [ ] 選択した馬はハイライト表示される
- [ ] 券種に応じた必要選択数が表示される（例：馬連は「2頭選択」）
- [ ] 必要な頭数が選択されるまでカート追加ボタンは無効
- [ ] 金額をプリセット（¥100, ¥500, ¥1,000, ¥5,000）または直接入力できる
- [ ] 認証不要で入力できる

---

## US-01-003a: 買い目のカート追加

**ストーリー**
> 競馬ファンとして、入力した買い目をカートに追加したい。なぜなら、複数の買い目をまとめてAIに相談したいから。

| 項目 | 内容 |
|------|------|
| 優先度 | Must have |
| 認証 | 不要 |
| 対象ペルソナ | 全ペルソナ |

**受け入れ条件**
- [ ] 「カートに追加」ボタンで買い目がカートに追加される
- [ ] 追加後、馬番選択がリセットされる
- [ ] 追加成功時にトースト通知が表示される
- [ ] ヘッダーのカートアイコンにバッジでアイテム数が表示される
- [ ] 異なるレースの買い目も同じカートに追加できる

---

## US-01-003b: カートの管理

**ストーリー**
> 競馬ファンとして、カート内の買い目を確認・編集・削除したい。なぜなら、AIに相談する前に買い目を整理したいから。

| 項目 | 内容 |
|------|------|
| 優先度 | Must have |
| 認証 | 不要 |
| 対象ペルソナ | 全ペルソナ |

**受け入れ条件**
- [ ] カート画面で全買い目の一覧が表示される
- [ ] 各買い目にレース名、券種、馬番、金額が表示される
- [ ] 各買い目に削除ボタンがある
- [ ] 「すべて削除」ボタンで一括削除できる（確認ダイアログ付き）
- [ ] 合計金額が表示される
- [ ] ログイン時は残り許容負け額が表示される

---

## US-01-003c: 複数買い目のまとめてAI相談

**ストーリー**
> 競馬ファンとして、カート内の複数の買い目をまとめてAIに相談したい。なぜなら、全体の賭け計画について客観的なフィードバックを得たいから。

| 項目 | 内容 |
|------|------|
| 優先度 | Must have |
| 認証 | 不要 |
| 対象ペルソナ | 全ペルソナ（特にペルソナ1） |

**受け入れ条件**
- [ ] 「まとめてAIに相談する」ボタンでAI相談画面に遷移する
- [ ] 全買い目の一覧が表示される
- [ ] 各買い目ごとにデータフィードバックが表示される
- [ ] 合計掛け金が表示される
- [ ] ログイン時は残り許容負け額と比較表示される
- [ ] 限度額超過時は購入ボタンが無効化される

---

## US-01-004: AIへの賭け相談

**ストーリー**
> 競馬ファンとして、入力した買い目についてAIに相談したい。なぜなら、衝動的に賭ける前に立ち止まって考える機会が欲しいから。

| 項目 | 内容 |
|------|------|
| 優先度 | Must have |
| 認証 | 不要 |
| 対象ペルソナ | 全ペルソナ（特にペルソナ1） |

**受け入れ条件**
- [ ] AI相談画面に遷移するとデータフィードバックが即時表示される
- [ ] 「立ち止まって考えましょう」のメッセージが表示される
- [ ] 投票締め切り時刻までの残り時間が表示される
- [ ] AIとの自由な会話（追加質問）が可能
- [ ] 認証不要で利用できる

---

## US-01-005: データに基づくフィードバック

**ストーリー**
> 競馬ファンとして、選んだ馬の過去成績、騎手情報、馬場状態、オッズなどのデータに基づくフィードバックを受けたい。なぜなら、客観的な情報をもとに判断したいから。

| 項目 | 内容 |
|------|------|
| 優先度 | Should have |
| 認証 | 不要 |
| 対象ペルソナ | 全ペルソナ（特にペルソナ2） |

**受け入れ条件**
- [ ] 選択した馬ごとに過去5走の成績が表示される
- [ ] 騎手の当該コースでの成績が表示される
- [ ] 現在の馬場状態に対する適性が表示される
- [ ] 現在のオッズと人気順が表示される
- [ ] 買い目ごとにカード形式で見やすく表示される
- [ ] 認証不要で利用できる

---

## US-01-006: 掛け金に対するフィードバック

**ストーリー**
> 競馬ファンとして、入力した掛け金額に対するフィードバックを受けたい。なぜなら、自分の賭けパターンや残り許容負け額を意識して適切な金額を決めたいから。

| 項目 | 内容 |
|------|------|
| 優先度 | Should have |
| 認証 | 不要（ログイン時は残り許容負け額も表示） |
| 対象ペルソナ | 全ペルソナ（特にペルソナ1） |

**受け入れ条件**
- [ ] 今回の合計掛け金額が表示される
- [ ] ログイン時は残り許容負け額が表示される
- [ ] ログイン時は過去の平均掛け金額との比較が表示される
- [ ] 掛け金が残り許容負け額を超える場合は警告が表示される
- [ ] 掛け金の調整を促すメッセージが表示される
- [ ] 未ログインでも基本的なフィードバックは表示される
