# 馬券会議 グランドデザイン

## プロダクト思想

> **「AIの力でユーザーの回収率を上げ、無謀な賭けを減らし、競馬を長く楽しめるようにする」**

このプロダクトは**ユーザーを勝たせる**ためのアプリである。

- 勝てる賭けを見つける → **YES**
- 期待値の低い賭けを止める → **YES**
- データに基づく推奨を提供 → **YES**
- 資金管理を教える → **YES**

「勝つための知識とツールを提供し、無謀な賭けから守る」を具現化するアプリである。

## デザイン4原則

### 原則1: 勝てる賭けを見つける

| 従来の競馬アプリ | 馬券会議 |
|-----------------|---------|
| 情報を並べるだけ | **勝てる馬を推奨** |
| ユーザー任せ | **AIが分析・提案** |
| オッズ表示のみ | **期待値を計算** |
| 買い目は自分で | **最適な買い目を提案** |

### 原則2: 無謀な賭けを止める

- 期待値の低い賭けは**明確に「やめろ」と言う**
- 根拠なく「なんとなく」は**データで否定**
- 負けを取り戻そうとする賭けは**警告**

### 原則3: 資金管理を教える

- 適切な賭け金額を提案
- 回収率・的中率を可視化
- 長期的な収支管理
- 「勝てる範囲」で賭けることを推奨

### 原則4: 長く楽しめるようにする

- 依存症になる賭け方は防ぐ
- 「一発逆転」より「コツコツ勝つ」
- 資金が尽きない賭け方を教える
- 競馬を長く楽しむための知識を提供

## AIの役割

### 「勝たせるAI」として

1. **レース分析**
   - 過去データから勝ち馬を予測
   - 期待値の高い買い目を提案
   - 「これは買い」「これはやめろ」を明確に

2. **資金管理アドバイス**
   - 適切な賭け金額を提案
   - 回収率100%超を目指す戦略
   - リスク分散の方法

3. **馬券術の教育**
   - オッズと期待値の関係
   - 人気薄の狙い方
   - 本命・穴のバランス

## カラーパレット

```css
:root {
  /* 基調色: 信頼・安定 */
  --color-primary-blue: #1e40af;      /* 信頼・安定・知性 */
  --color-primary-blue-light: #dbeafe; /* 背景・選択状態 */

  /* 成功色: 推奨・期待値高 */
  --color-success-green: #16a34a;     /* 推奨買い目 */
  --color-success-green-light: #dcfce7;

  /* 警告色: 注意喚起（期待値低・リスク高） */
  --color-alert-amber: #f59e0b;       /* 期待値低めの賭け */
  --color-alert-red: #dc2626;         /* 推奨しない賭け・限度額到達 */

  /* 中立色: データの客観的表示 */
  --color-neutral-50: #f9fafb;
  --color-neutral-200: #e5e7eb;
  --color-neutral-500: #6b7280;
  --color-neutral-800: #1f2937;
}
```

**カラー使用ガイド**:
- 緑: AIが推奨する買い目、期待値が高い
- 青: ニュートラル、情報表示
- 黄: 注意が必要、期待値やや低い
- 赤: 推奨しない、やめておくべき

## 常時表示すべき要素

### 1. 負け額限度額バー（将来実装予定）

すべての画面で常に表示される「限度額バー」

- 通常時: 緑
- 80%到達: 黄色（警告）
- 100%到達: 赤（購入停止）

> **Note**: ログイン機能実装後に対応予定。現在はカート画面にモックを表示。

### 2. 依存症相談窓口リンク

すべてのページで**ボトムナビの上**に常時表示

```
困ったときは｜ギャンブル依存症相談窓口 →
```

- 目立ちすぎない（アプリの邪魔にならない）
- しかし、必要なときに**すぐ見つけられる**

| 推奨馬券の的中率 | **測定・改善** |
| 推奨馬券の回収率 | **100%以上を目指す** |
| 無謀な賭けの抑制率 | **測定・改善** |
| 負け額限度額遵守率 | **100%**（システム強制） |

## やること・やらないこと

### やること
- 勝てる馬・買い目の推奨
- 期待値計算
- 資金管理アドバイス
- 無謀な賭けへの警告
- 長期的な収支管理

### やらないこと
- 「絶対当たる」という煽り
- 過度な期待を持たせる表現
- 負けを取り戻させる誘導
- 賭け金を増やす方向への誘導
