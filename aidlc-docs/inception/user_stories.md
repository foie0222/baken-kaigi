# EPIC-01: AI対話機能

## US-01-001: レース一覧の表示

**ストーリー**
> 競馬ファンとして、当日開催されるレースの一覧を確認したい。なぜなら、どのレースに賭けるか検討したいから。

| 項目 | 内容 |
|------|------|
| エピック | EPIC-01: AI対話機能 |
| 優先度 | 高 |
| 認証 | 不要 |
| 対象ペルソナ | 全ペルソナ |

**受け入れ条件**
- [ ] 当日開催の全レースが一覧表示される
- [ ] 各レースにレース番号、レース名、発走時刻が表示される
- [ ] 開催場（東京、中山、京都など）でフィルタできる
- [ ] 日付を選択して別日のレースも閲覧できる
- [ ] レースをタップすると詳細画面に遷移する
- [ ] 認証不要で閲覧できる

## US-01-003: 買い目の入力

**ストーリー**
> 競馬ファンとして、購入したい馬券の買い目（馬番、賭け式、金額）を入力したい。なぜなら、AIに相談する前に自分の賭け内容を明確にしたいから。

| 項目 | 内容 |
|------|------|
| エピック | EPIC-01: AI対話機能 |
| 優先度 | 高 |
| 認証 | 不要 |
| 対象ペルソナ | 全ペルソナ |

**受け入れ条件**
- [ ] 券種（単勝、複勝、馬連、馬単、三連複、三連単）を選択できる
- [ ] 馬番をチェックボックスで選択できる
- [ ] 選択した馬はハイライト表示される
- [ ] 券種に応じた必要選択数が表示される（例：馬連は「2頭選択」）
- [ ] 必要な頭数が選択されるまでカート追加ボタンは無効
- [ ] 金額をプリセット（¥100, ¥500, ¥1,000, ¥5,000）または直接入力できる
- [ ] 認証不要で入力できる

**設計上の考慮事項**
- 馬番はチェックボックスで選択可能（馬リストから直接タップ/クリック）
- 券種に応じて必要な選択数を表示（単勝:1頭、馬連:2頭、三連複:3頭など）
- 必要な頭数が選択されるまでカート追加ボタンは無効化
- 選択した馬番は視覚的にハイライト表示

## US-01-011: 期待値の表示

**ストーリー**
> 競馬ファンとして、自分の買い目の期待値を知りたい。なぜなら、長期的に勝てる賭けかどうか判断したいから。

| 項目 | 内容 |
|------|------|
| エピック | EPIC-01: AI対話機能 |
| 優先度 | 高 |
| 認証 | 不要 |
| 対象ペルソナ | 全ペルソナ |

**受け入れ条件**
- [ ] 買い目入力画面で選択した買い目の期待値が表示される
- [ ] 期待値は「100%以上」「80-100%」「80%未満」で色分け表示される
- [ ] 期待値100%以上は緑色で「推奨」表示
- [ ] 期待値80%未満は赤色で「非推奨」表示
- [ ] 期待値の計算根拠（AI予測確率 × オッズ）が表示される
- [ ] 認証不要で利用できる

**設計上の考慮事項**
- 期待値 = AI予測確率 × オッズ × 0.8（控除率考慮）
- 期待値100%以上 = 長期的に利益が出る可能性
- 期待値が低い買い目は明確に「やめた方がいい」と伝える

|------|------|
