# 馬券会議 デザイン改善提案書 v2

## 1. プロダクト思想の理解

### 1.1 核心的ミッション

> **「依存症傾向のあるユーザーが、AIとの対話を通じて立ち止まって考え、自己設定した負け額限度額内で責任ある賭けを行える」**

このプロダクトは**競馬予想アプリではない**。

- 勝ち方を教えるアプリ → **NO**
- 衝動的な購入を促すアプリ → **NO**
- 「当たる」を煽るアプリ → **NO**

「立ち止まって、考えましょう」を具現化するアプリである。

### 1.2 最優先ペルソナ

**田中 健一（42歳）**
- 競馬歴15年、月間10〜30万円（変動大）
- 負けを取り戻そうと賭け金が増加傾向
- 「やめたい」とは思っていないが、**「コントロールしたい」**と強く感じている
- 一人で賭けていると冷静な判断ができなくなる
- 家族に心配をかけたくない

### 1.3 成功指標

| 指標 | 目標 |
|------|------|
| 負け額限度額遵守率 | **100%**（システム強制） |
| 負け額減少率 | 測定・改善 |
| AI対話実施率 | **80%以上** |

**社会的ミッション: ユーザーの負け額を減らすこと**

## 2. デザイン原則（プロダクト思想に基づく）

### 原則1: 煽らない

```
❌ 一般的な競馬アプリ          ✅ 馬券会議
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
「今すぐ買え！」              「立ち止まって、考えましょう」
派手な色・アニメーション       落ち着いた色・静かな動き
大きなCTAボタン               控えめなアクション
緊急感を煽る                  余裕を与える
オッズを強調                  データを淡々と提示
```

### 原則2: 立ち止まらせる

- 購入ボタンを押す前に**必ずAI相談を経由**
- 衝動的なタップを防ぐ**適度な摩擦**
- 「やめておく」という選択肢を**常に用意**

### 原則3: 味方であることを示す

- AIは「煽る存在」ではなく「寄り添う存在」
- 「あなたの代わりに考える」ではなく「あなたと一緒に考える」
- 責める口調ではなく、共感する口調

### 原則4: 現実を直視させる

- 損益を**隠さず、ごまかさず**表示
- 負け額限度額を**常に意識させる**
- 「いくら使ったか」を**客観的に提示**

## 4. 改善提案

### 4.1 カラーパレットの意図的設計

**現状は正しい方向性**。ただし、より意図的に使い分ける。

```css
:root {
  /* === 基調色: 落ち着き・安心 === */
  --color-calm-green: #1a5f2a;      /* 信頼・安定・自然 */
  --color-calm-green-light: #e8f5ea; /* 背景・選択状態 */

  /* === 警告色: 注意喚起（煽りではなく気づき） === */
  --color-alert-amber: #f59e0b;     /* 限度額接近時 */
  --color-alert-red: #dc2626;       /* 限度額到達時 */

  /* === 中立色: データの客観的表示 === */
  --color-neutral-50: #f9fafb;
  --color-neutral-200: #e5e7eb;
  --color-neutral-500: #6b7280;
  --color-neutral-800: #1f2937;

  /* === AI: 寄り添い・信頼 === */
  --color-ai-primary: #6366f1;      /* 落ち着いた紫/青 */
  --color-ai-background: #eef2ff;
}
```

**注意**: 赤・オレンジは「購入を促す」ためではなく、「危険を知らせる」ためにのみ使用。

### 4.3 「やめておく」ボタンのデザイン

**現状:**
```css
.btn-secondary {
  background: #f5f5f5;  /* 目立たない */
  color: #333;
}
```

**改善案:**
```css
/* 「やめておく」は正しい判断を讃える色 */
.btn-stop {
  background: var(--color-calm-green);
  color: white;
  padding: var(--space-4) var(--space-6);
  border-radius: var(--radius-lg);
  font-size: var(--font-size-lg);
  font-weight: 600;
  width: 100%;
}

/* 「購入する」は慎重さを求める控えめなスタイル */
.btn-purchase {
  background: transparent;
  color: var(--color-neutral-500);
  padding: var(--space-3);
  font-size: var(--font-size-sm);
  text-decoration: underline;
  border: none;
}
```

**心理的効果:**
- 「やめておく」を選ぶことが**ポジティブな行動**だと感じさせる
- 「購入する」は**慎重に選ぶべき選択肢**だと伝える

### 4.5 AIのキャラクターデザイン

**現状:**
- 🤖 ロボット絵文字
- 「馬券会議 AI」という名前
- 機械的な印象

**改善案:**

AIは「煽る存在」ではなく「寄り添う存在」であるべき。

```
┌────────────────────────────────────────┐
│                                        │
│    ┌──────┐                            │
│    │  ◯   │   馬券会議 AI              │
│    │      │   あなたと一緒に考えます   │
│    └──────┘                            │
│                                        │
└────────────────────────────────────────┘
```

**キャラクター設計:**

| 要素 | 現状 | 改善案 |
|------|------|--------|
| アイコン | 🤖 ロボット | 穏やかな丸いアイコン（抽象的・非人間的） |
| 色 | 緑グラデーション | 落ち着いた紫/青（`#6366f1`） |
| 口調 | 「〜ですか？」 | 「〜かもしれませんね」「一緒に考えてみましょう」 |
| サブテキスト | 「立ち止まって、考えましょう」 | そのまま維持（非常に良い） |

**なぜロボット絵文字を避けるか:**
- 「機械に指示される」印象を与える
- 距離感が生まれる
- 抽象的なシンボルの方が「味方」として受け入れやすい

## 5. 改善しない方が良い点

前回の提案で「改善」として挙げた以下の項目は、**プロダクト思想に反するため採用しない**:

| 前回の提案 | 採用しない理由 |
|-----------|---------------|
| CTAボタンを目立たせる | 購入を煽ることになる |
| カードに立体感・影を強調 | 視覚的刺激を増やす必要はない |
| ゴールドアクセントで華やかに | 競馬アプリ的な煽りになる |
| ホバー効果を派手に | 衝動的クリックを誘発する |
| アイコンライブラリで洗練 | 現状の絵文字で十分（親しみやすさ） |

**このアプリは「かっこいいUI」を目指すものではない。**

## 7. まとめ

```

「コンバージョン向上」         「衝動的購入の抑制」
「ユーザーを惹きつける」       「ユーザーを守る」
```

### 忘れてはいけないこと

> 最優先ペルソナは「田中 健一（42歳）」
>
> 競馬歴15年、月間10〜30万円（変動大）
>
> **「やめたい」とは思っていないが、「コントロールしたい」と強く感じている**
>
> 一人で賭けていると冷静な判断ができなくなる

このプロダクトのデザインは、**田中さんが冷静でいられるため**に存在する。

煽らない。急かさない。寄り添う。現実を見せる。

それが馬券会議のデザイン哲学である。
